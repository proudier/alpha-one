# Alpha-One

A Jekyl theme based on Bootstrap.


## Installation

Add this line to your Jekyll site's Gemfile:

```ruby
gem "alpha-one"
```

And add this line to your Jekyll site:

```yaml
theme: alpha-one
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install alpha-one
